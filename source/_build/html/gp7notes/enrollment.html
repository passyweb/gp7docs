Inscriptions à une session et aux cours¶

Le menu Inscriptions permet de gérer tous les inscriptions aux cours et à une session d’examens. Une inscription à une session d’examens se déroule en deux étapes : Une inscription aux cours et une inscription à la session.

Inscrire à une session d’examens : Cette opération permet d’inscrire aux cours à la session ainsi aux cours liés à une période d’enseignement

  • Aller dans la menu Inscriptions, cliquer sur Inscrire/désinscrire.
  • Dans la fenêtre qui apparait, cliquer sur Inscrire, dans la liste déroulante qui apparait sélectionner Tous, puis cliquer sur Afficher.
  • Sélectionner les différents étudiants à inscrire, puis cliquer sur Inscrire.

Gérer les inscriptions aux cours : Cette opération permet gérer que les inscriptions aux cours liés à la session d’examens en cours. Pour gérer les inscriptions aux cours sur l’ensemble, aller dans Gp7.

Deux possibilités sont offertes : la gestion par étudiant ou la gestion par cours.

Gérer les inscriptions aux cours par étudiant

  • Aller dans le menu Inscriptions, puis cliquer sur Gérer les inscriptions aux cours par étudiant.
  • Dans la fenêtre qui apparait, cliquer le bouton Etudiant par sélectionner un étudiant.
  • Une nouvelle fenêtre apparait, entrer le nom d’un étudiant dans le champs Nom ou Prénom, puis cliquer sur Rechercher. Pour lister tous les étudiants, laisser le vide puis cliquer sur Rechercher
  • Cliquer sur un nom pour le sélectionner puis cliquer sur Gérer. La fenêtre se ferme pour lister les différents cours auxquels l’étudiant est inscrit.
  • Pour ajouter un cours, cliquer sur Ajouter. Dans la fenêtre qui apparait, cocher les cours voulus et puis cliquer sur OK, puis valider le message de confirmation
  • Pour supprimer un cours (désinscrire ou dispenser à un cours), cocher les cours à supprimer, puis cliquer sur Supprimer

NB : Pour ajouter (inscrire) un cours de complément : Dans la fenêtre de sélection des cours, cocher Compléments, puis sélectionner un ensemble de cours pour lister les cours.

Gérer les inscriptions aux cours par cours

  • Aller dans le menu Inscriptions, puis cliquer sur Gérer les inscriptions aux cours par cours
  • Dans la fenêtre qui apparait, cliquer sur le bouton Cours pour sélectionner un cours.
  • Cocher Inscrire pour inscrire aux cours ou Désinscrire pour dispenser à un cours
  • Cocher les étudiants à inscrire puis cliquer sur Valider.
